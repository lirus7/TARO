This is a basic login portal created using node.js and MongoDB as a database.

You can see the screenshots for the resultant output

To run this project :

pre-reqs:
1.Should have node.js,npm and MongoDB installed with the latest versions .

2.Fork/Download the folder

3.Extract it and run npm install in the terminal.

4.Run mongod in a separate terminal to have mongodb running in the background so the db in the webapp can access the database

5.Run node server.js or nodemon server.js
